# CMS Web Portofolio (React + TSX)

Minimal CMS with Login and Dashboard that consumes the Go API.

- Dev: npm install; npm run dev (default port 5174)
- API proxy: /api -> http://localhost:8080

Pages:
- / (Login)
- /dashboard (Admin dashboard; shows view stats)

Configure backend CORS to allow http://localhost:5174 (already added).
